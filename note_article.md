# Excelのセル内改行を、もっと自然に。「いつもの改行 for Excel」を作りました

Excelでセルの中に改行を入れるとき、通常は `Alt + Enter` を押します。

慣れてしまえばできる操作ではありますが、文章を入力している途中で毎回Altキーを添えるのは、地味にテンポが崩れます。

特に、メモ、議事録、コメント、作業メモのように、Excelのセル内で何度も改行したい場面では、この小さな操作が少しずつ面倒になります。

## こんなことありませんか？

TeamsのチャットやChatGPTでは、改行したいときに `Shift + Enter` を押すことがあります。

その感覚のままExcelのセル内で `Shift + Enter` を押してしまい、なぜか一つ上のセルに移動してしまう。

あるいは、セル内で改行しようと思って間違えて `Enter` を押して、下のセルへ移動し、そのまま入力を続けて下のセルの内容を上書きしてしまう。

Excelだけ改行の操作が少し違うので、他のアプリの感覚で入力していると、こういう小さな事故が起きがちです。

そこで、`Enter` または `Shift + Enter` でセル内改行できるWindows常駐アプリを作りました。

名前は **いつもの改行 for Excel** です。

英語表示では **LineBuddy for Excel** という名前にしています。

## 何ができるアプリ？

「いつもの改行 for Excel」は、Excelでセル内に文字を入力しているときだけ、選択したキー入力を `Alt + Enter` に変換する常駐アプリです。

選べるセル内改行キーは2つです。

- `Enter`
- `Shift + Enter`

たとえば `Enter` を選ぶと、Excelのセル内で文字入力中にEnterを押すだけで、セル内改行になります。

`Shift + Enter` を選ぶと、Shift + Enterだけがセル内改行になります。

## セル内入力中だけ動きます

このアプリで大事にしたのは、「変換してよい場面だけで動くこと」です。

セルを選択しているだけの状態でEnterを押した場合は、Excel本来の動きどおり、下のセルへ移動します。

また、Excel上の図形やテキストボックスの中で文字を入力しているときも、変換しないようにしています。

つまり、基本的には次の状態だけで動きます。

**Excelのセル内で文字入力中のとき**

それ以外では、誤変換を避けるために動作を止める設計です。

## 現在の状態が分かるようにしました

アプリのダイアログには、現在の状態をステータスとして表示します。

表示される主なステータスは次のとおりです。

- `ステータス：セル内入力中`
- `ステータス：セル内入力中ではありません`
- `ステータス：セル内入力中を確認できません`
- `ステータス：一時停止中`

「セル内入力中」のときだけ、選択したキーを `Alt + Enter` に変換します。

「セル内入力中ではありません」や「セル内入力中を確認できません」のときは、安全側に倒して変換しません。

## 判定のしくみ

このアプリは、次のような情報を組み合わせて、セル内入力中かどうかを判定しています。

- Excelが前面にあるか
- 図形・テキストボックスを操作していないか
- Excelの選択状態がセル範囲か
- 文字カーソルやExcelの編集状態が確認できるか
- 直前に文字入力、F2、ダブルクリックなどの編集開始操作があったか

Excelのバージョンによって取得できる情報が違うため、複数の手がかりを組み合わせています。

ただし、判断材料が足りないときは無理に変換しません。

「うまく変換されないことがある」よりも、「変換してはいけない場所で誤変換すること」の方が困るためです。

## 日本語入力にも配慮しました

`Enter` をセル内改行キーにすると、日本語入力では少し注意が必要です。

日本語変換中のEnterは、まず変換確定を優先します。

たとえば、

1. 日本語を入力する
2. Spaceで変換する
3. Enterで変換を確定する
4. もう一度Enterでセル内改行する

という動きになります。

これは通常の `Alt + Enter` でセル内改行する場合と同じように、日本語入力の確定を優先するためです。

変換確定とセル内改行が同時に起きると入力しづらいため、このような挙動にしています。

## 既存文字セルを編集するときの安全設計

すでに文字が入っているセルを編集モードにした直後は、環境によって、選択したキーがセル内改行にならないことがあります。

これは、図形やテキストボックスでの誤変換を避けるための安全設計です。

1文字でも入力した後は、選択したキーでセル内改行できます。

## タスクトレイに常駐します

アプリを起動すると、タスクトレイに常駐します。

右クリックメニューから、次の操作ができます。

- ダイアログを開く
- 一時停止 / 再開
- Windows起動時に実行する
- Windows起動時の登録を解除する
- 日本語 / English の切り替え
- バージョン情報を見る
- 終了する

一時停止中は、タスクトレイのアイコンもモノクロになります。

## 日本語 / 英語を切り替えできます

アプリ本体は、1つのexeで日本語と英語に対応しています。

初回起動時はWindowsの表示言語を見て、日本語または英語を自動で選びます。

あとから変更したい場合は、ダイアログ右上の `English` / `日本語` ボタン、またはタスクトレイの右クリックメニューから切り替えできます。

英語表示でのアプリ名は **LineBuddy for Excel** です。

## 対応環境について

Windows向けのアプリです。

動作確認は次のExcelで行っています。

- Excel 2010
- Excel 2019
- Microsoft 365版Excel

ただし、Excelはバージョンや更新状況によって、セル編集中の判定方法が変わることがあります。

そのため、一部環境ではうまく動作しない可能性があります。

このあたりは、Excelの内部的な挙動に左右される部分があり、できるだけ誤動作しないように、安全側の判定にしています。

## こんな人におすすめです

- Excelでメモや文章をよく入力する
- セル内改行をよく使う
- `Alt + Enter` が少し面倒に感じる
- `Enter` や `Shift + Enter` で自然に改行したい
- TeamsやChatGPTの入力感覚に近づけたい
- Excelでの入力テンポを少しでも崩したくない
